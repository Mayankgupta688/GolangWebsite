Maintaining Previous State / Props in React Hooks using “useRef”

Use “useRef” to preserve the previous value of State / Props

TechnoFunnel presents another article which helps you to understand How to retain previous State or Props value in React. While the React component is re-rendering, user might require to save previous state or props values. We can use useRef Hook in order to preserve the Previous State or Props Value or save some persistent data accross multiple re-rendering of the component


Understanding React “useRef” Hook

Understanding React “useRef” Hook

Lets see how to use “useRef” with the help of Example below:



In the above URL, we are creating “userName” variable using useRef. This variable now act as a global/common data for the component. We are then rendering the component with the “userName.current” on the screen. Once the component is Initially rendered with the “userName” as “Mayank Gupta”, useEffect is called which updates the variable to “TechnoFunnel”. Now everytime the component re-renders the value of “userName” will be “TechnoFunnel”. The useRef data is persistent accross multiple rendering.
